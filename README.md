# Project Overview

This project presents a **read-only API Security Risk Analysis** performed on the **JSONPlaceholder** public demo API.  
The objective of this project is to identify common API security risks, understand their potential business impact, and recommend appropriate remediation measures.

The assessment is conducted strictly for **educational and learning purposes**, following ethical and legal guidelines.  
No exploitation or intrusive testing was performed.
##  Project Objectives

- Analyze a public/demo REST API  
- Identify common API security risks  
- Assess authentication and authorization controls  
- Understand business impact of insecure APIs  
- Provide clear remediation recommendations  
- Practice thinking like a security consultant  

---

##  API Details

- **API Name:** JSONPlaceholder  
- **API Type:** Public Demo REST API  
- **Base URL:** https://jsonplaceholder.typicode.com  
- **Purpose:** Provides fake data for testing and learning API development  

### Endpoints Tested:
- `/users`  
- `/users/{id}`  
- `/posts`
